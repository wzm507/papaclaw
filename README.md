# 胶囊潮流·AI出海服务官网

AI赋能中国企业出海赢全球

## 项目结构

```
├── client/          # 前端代码
├── server/          # 后端代码
├── shared/          # 共享代码
└── dist/            # 构建产物
```

## 技术栈

- React + TypeScript
- NestJS
- TailwindCSS
- Vite

## 运行项目

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
